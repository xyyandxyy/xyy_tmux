#  XYY DIY TMUX 配置

## 简介

这是xyy DIY的tmux配置, based on `Oh my tmux!`

## 安装方法

安装要求：
- tmux 2.6或更高版本


安装步骤：

```
cd ~
git clone --single-branch https://github.com/xyyandxyy/xyy_tmux.git
ln -s -f ~/xyy_tmux/.tmux.conf ~/.tmux.conf
ln -s -f ~/xyy_tmux/.tmux.conf.local ~/.tmux.conf.local
```

## Session / Window / Pane 的区别

tmux 有三层结构，从外到内依次是：

```
Session  =  一个"工作项目"
  └── Window  =  项目里的一个"标签页"
        └── Pane  =  标签页里分割出来的"小格子"
```

**Session** 是最外层容器。detach 之后 session 还在后台活着，里面的程序继续跑，重新 attach 回来一切还在。适合按**项目**来隔离。

**Window** 类似浏览器标签页，每个 window 占满整个终端屏幕，底部状态栏会显示所有 window。适合按**任务类型**区分，比如一个 window 写代码，一个 window 看日志。

**Pane** 是把一个 window 切割成多个小格子，**同时可见**。适合需要同时盯着多个内容的场景，比如左边写代码右边看运行结果。

```
┌─────────────────────────────────────┐
│  Session: "工作项目"                  │
│  ┌─────────────────────────────────┐ │
│  │  Window 1: 代码    Window 2: 日志│ │  ← 底部状态栏
│  │ ┌──────────┬──────────────────┐ │ │
│  │ │          │                  │ │ │
│  │ │  Pane 1  │     Pane 2       │ │ │  ← 同时可见
│  │ │  vim     │  npm run dev     │ │ │
│  │ │          │                  │ │ │
│  │ └──────────┴──────────────────┘ │ │
│  └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

| 场景 | 用什么 |
|------|--------|
| 换了一个完全不同的项目 | 新建 **Session** |
| 同一项目，切换到不同任务（写代码 vs 查日志） | 新建 **Window** |
| 需要同时看到两个内容（代码 + 输出） | 分割 **Pane** |
| 要离开电脑但程序要继续跑 | Detach **Session**（`` ` + d``） |

## 常用快捷键

> 快捷键说明：`` ` `` 是前缀键（反引号）；`control` 是 Mac 键盘左下角的 `control` 键（不是 `⌘ command`）

- `` ` + e`` 编辑本地配置文件
- `` ` + r`` 重新加载配置

### Session 管理
- `` ` `` 然后按住 `control + c` — 创建新会话
- `` ` `` 然后按住 `control + f` — 按名称切换会话
- `` ` + $`` 重命名当前会话
- `` ` + s`` 显示会话列表并选择
- `` ` + d`` 分离(detach)当前会话
- `` ` + D`` 选择要分离的会话
- `` ` + )`` 切换到下一个会话
- `` ` + (`` 切换到上一个会话

### Window 管理
- `` ` + c`` 创建新窗口
- `` ` + [`` 切换到上一个窗口
- `` ` + ]`` 切换到下一个窗口
- `` ` + 数字`` 切换到指定编号的窗口
- `` ` + w`` 显示窗口列表并选择
- `` ` + ,`` 重命名当前窗口
- `` ` + &`` 关闭当前窗口
- `` ` + .`` 修改当前窗口编号

### Pane 管理
- `` ` + -`` 水平分割窗格（上下）
- `` ` + =`` 垂直分割窗格（左右）
- `` ` `` 然后按住 `control + h/j/k/l` — 在窗格间移动（左/下/上/右）
- `` ` ``+ ``方向键`` 在窗格间移动
- `` ` + h/j/k/l`` 调整窗格大小（左/下/上/右）
- `` ` + +`` 将当前窗格最大化到新窗口
- `` ` + z`` 最大化/还原当前窗格
- `` ` + q`` 显示窗格编号
- `` ` + {`` 将当前窗格移到左边
- `` ` + }`` 将当前窗格移到右边
- `` ` + x`` 关闭当前窗格
- `` ` + !`` 将当前窗格变成一个新窗口
- `` ` + 空格`` 循环切换窗格布局

### 复制模式和鼠标操作
- `` ` + m`` 切换鼠标模式
- `` ` + Enter`` 进入复制模式
- `` ` + p`` 从剪贴板粘贴
- 复制模式下：
  - `v` 开始选择
  - `y` 复制选中内容
  - `Esc` 退出复制模式
  - 鼠标拖选自动复制

### 其他实用功能
- `` ` + :`` 进入命令模式
- `` ` + ?`` 显示所有快捷键
- `` ` + ~`` 显示信息面板
- `` ` + t`` 显示时钟

## 配置文件说明

### `.tmux.conf`
这是来自 `Oh my tmux!` 的主配置文件，包含了所有核心逻辑和默认行为。**不建议直接修改这个文件**，否则之后更新上游配置时会产生冲突。

### `.tmux.conf.local`
这是个人自定义配置文件，所有个性化改动都写在这里。`.tmux.conf` 在加载时会自动读取这个文件，本地设置会覆盖默认值。

本配置在 `.tmux.conf.local` 里做的主要改动有：
- 前缀键改为 `` ` ``（反引号）
- 用 `=` 和 `-` 分割窗格（垂直/水平）
- 用 `h/j/k/l` 调整窗格大小，用 `control+h/j/k/l` 在窗格间移动
- 用 `[` 和 `]` 切换上/下一个窗口
- 切换鼠标模式时会在状态栏显示提示信息

使用 `` ` + e`` 可以快速打开 `.tmux.conf.local` 进行编辑，保存后会自动重新加载配置。

主要可配置项：
- `tmux_conf_theme_*` 系列变量：控制主题和颜色
- `tmux_conf_*` 系列变量：控制各种行为设置
